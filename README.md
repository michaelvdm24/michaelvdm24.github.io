# michaelvdm24.github.io

Personal portfolio site for Michael van der Merwe, hosted on GitHub Pages.

Built with plain HTML, CSS, Bootstrap 4, jQuery, and AOS animations. No server-side dependencies.

## Structure

```
index.html          — main portfolio page
projects/           — individual project detail pages
static/css/         — stylesheets
static/img/         — images and GIFs
static/scripts/     — JavaScript and JSON data
favicon.ico
```

## Local preview

```bash
python -m http.server 8000
```

Then open `http://localhost:8000` in a browser.
